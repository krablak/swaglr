{% extends "based.html" %} {% block cliplist %} What is it?

Swag Clips is only the bag of posted web clips... nothing more.

How to use it?

1. Login to the Swag Clips.
2. Download and install the Swag Clip Chrome extension.
3. Perform first clip swag to authorize exetension using the Chrome context menu.
4. Swag a text, link or image clip.

How it works?

Whole Swag Clips server-side is running on Google App Engine and is written in Python. Web pages and browser extension is build and tested only for Chrome 6.
Authentication between server application and browser extension is based on OAuth.
